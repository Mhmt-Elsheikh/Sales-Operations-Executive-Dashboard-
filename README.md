Sales & Operations Executive Dashboard

Description: Built a comprehensive interactive dashboard entirely in Excel to monitor cross-regional sales performance. Utilized Pivot Tables and Slicers to enable dynamic filtering by Region, Shipping Method, and Product Category. The tool visualizes revenue trends over 4 years and analyzes logistics efficiency, empowering quick operational decision-making.

Tech Stack: Advanced Excel, Pivot Tables, Power Query, Data Visualization.
